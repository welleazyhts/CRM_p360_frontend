System Overview

PY360 is an enterprise-grade Customer Relationship Management (CRM) system designed for insurance renewals, lead management, customer service, and comprehensive business operations. Built with React 18+, Material-UI, and supporting 13 Indian languages.

1. AUTHENTICATION & SECURITY

This section covers all authentication and security features of the PY360 CRM system. It includes comprehensive user login functionality, multi-factor authentication (MFA) with OTP verification, user registration with validation, profile management capabilities, and a robust role-based access control system. The platform supports 7 distinct user roles (admin, client_admin, manager, renewals_specialist, all_modules_manager, agent, and viewer) with 8 permission groups covering renewals, email, business, marketing, survey, whatsapp, admin, and core functionalities. Security features ensure data protection and compliance with enterprise requirements.

4. CUSTOMER MANAGEMENT MODULE

The Customer Management Module offers a complete 360-degree view of customers throughout their relationship with the organization. It includes a contact database for managing all contacts with search, filtering, grouping, and import/export capabilities.
